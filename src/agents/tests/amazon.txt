- **Components**:

## Grid & Spacing
- **Spacing**: 
  - Padding and margins around each section for visual separation
  - Uniform gutter space between grid items
- **Alignment**: 
  - Centered alignment for text within grid items

## Typography
- **Font Families**: Default Amazon font
- **Font Sizes**: 
  - Headers: Large, bold
  - Sub-text: Medium, regular
  - Links: Medium, underline
- **Text Alignment**: Left-aligned for most text, center for main banner

## Colors
- **Primary Colors**: 
  - Background: White
  - Text: Black
- **Accent Colors**: 
  - Banner: Blue with white text
  - Links: Blue

## Images and Media
- **Dimensions**: Consistent square aspect ratio for grid images
- **Styling**: 
  - Rounded corners for images
  - Placeholder links for images

## Interactive Elements
- **Buttons**: 
  - Hover effect: Darken background
  - Active states: Light shadow
- **Links**: Underline on hover

## Iconography and Graphics
- **Icons**: 
  - Cart and account icons in the header
- **Style**: Simple, flat design

## Accessibility
- **Contrast Ratio**: Ensure text is legible against backgrounds
- **Alt Text**: Provide descriptive alt text for images
- **Focus States**: Clear focus indicators for interactive elements
